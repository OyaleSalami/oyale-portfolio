Project Overview

A CAM (Computer-Aided Manufacturing) software developed for a client. This project involved writing an interpreter for GCode and interfacing that with a simulation in Unity, allowing users to visualize and control CNC machining processes.

Key Features

  • GCode parsing and interpretation
  • Real-time simulation in Unity
  • User interface for loading and editing GCode files
  • Visualization of machining paths

Technologies Used

Unity for the simulation engine, C# for scripting, and custom parsing logic for GCode. Integrated with desktop application frameworks for the UI.

Challenges

Implementing accurate GCode interpretation required deep understanding of CNC standards. Optimizing the simulation for real-time performance was crucial.
